Eclipse Integration Build 20020129

What's new in this drop

  • A new attribute called showScopeSection has been added to the markup language for Search pages. If this attribute is missing or set to false then the scope section is not shown in the Search dialog. To see the scope section this attribute has to be set to true . Plug-ins which add their own Search page and want to see the scope section will have to add the new attribute to their plugin.xml

API changes

  • Activation of the Search view has been moved from the dialog to the individual Search page. Plug-ins which want the Search page to appear when Search is invoked have to call SearchUI.activateSearchResultView()
